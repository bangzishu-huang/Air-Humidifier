# Air-Humidifier
An ultrasonic air humidifier that will utilize PET bottles and also an atomiser module in order to humidify dry places.

This was a fun little side project I wanted to try out and see what I could make. It was fun designing the case that encapsulates the atomiser module and I can't wait to have this in my room!

# CAD Assembly
All of the 3d design work was done on a software called AutoDesk Inventor (I high recommend (:<)

# Firmware
The firmware for this project was relatively simple. It was code to ensure that the button for the atomiser module acts as an on and off switch. The application used to input this code into is Arduino IDE.
